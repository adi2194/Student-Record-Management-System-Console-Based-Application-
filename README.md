# Student-Record-Management-System-Console-Based-Application-
This program is a console-based Student Record Management System written in Python.
It stores student data using a list of dictionaries.
Each student dictionary contains name, roll number, marks, total, average, and grade.
The program uses functions to organize different operations like adding students,
viewing records, searching, and calculating statistics.
A while loop is used to continuously display the menu until the user selects Exit.
A for loop is used for searching and displaying students.
Grades are calculated using conditional statements based on average marks.
Duplicate roll numbers are prevented using a validation check.
The program demonstrates concepts of lists, dictionaries, loops, and functions.
AI tools were used for structure guidance, but the logic and understanding were verified.
